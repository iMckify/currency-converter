## Steps to start project in **Linux**:
1. install java 8 and node ^v16.13.2 with npm ^8.1.2
2. cd currency-converter
3. cd GUI
4. npm install
5. npm run linux
6. npm start
7. cd ..
8. java -jar backend.jar
9. open http://localhost:3000/ in browser
 
## Steps to start project in **Windows 10**:
1. install java 8 and node ^v16.13.2 with npm ^8.1.2
2. cd currency-converter
3. cd GUI
4. npm install
6. npm run startWin
7. cd ..
8. java -jar backend.jar
9. open http://localhost:3000/ in browser
